# 在线更新
- 注释:插件只对应本人个人的固件，因为不同人的固件的下载url地址不同，变量不同，以及本地版本号文件和远程拉取的版本号对比文件都不同(这是在编译固件时个人源码生成)，所以直接fork是无法正常使用的。
